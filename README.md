# MikroE NECTO Demos

## About

> This repository holds all DEMO packages used in [NECTO Studio](https://www.mikroe.com/necto).
>
> The DEMO packages shall serve as a usage outline for [NECTO Studio](https://www.mikroe.com/necto) related features.
